# Development Roadmap

This document outlines the future development plans for the EVE-KILL project.

## Short Term

-   **Complete Documentation:** Finish filling out all the `TODO` sections in the documentation.
-   **Improve Test Coverage:** Increase the test coverage for the application.
-   **Refine UI:** Improve the user interface and user experience.

## Medium Term

-   **Add More API Endpoints:** Expand the API to provide more data to clients.
-   **Improve Performance:** Optimize database queries and other performance-critical parts of the application.
-   **Add Theming:** Allow users to customize the look and feel of the application.

## Long Term

-   **Add a Plugin System:** Allow developers to extend the functionality of the application with plugins.
-   **Federation:** Allow multiple instances of the application to share data with each other.

*This roadmap is a work in progress and is subject to change.*
