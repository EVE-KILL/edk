# EVE-KILL Documentation

Welcome to the official documentation for EVE-KILL, a high-performance, self-hosted EVE Online killboard. This documentation is designed to help you understand, develop, and operate the EVE-KILL application.

Whether you're a developer looking to contribute, an administrator setting up your own instance, or just curious about how it all works, you've come to the right place.

## Getting Started

If you're new to EVE-KILL, we recommend starting with the [**Architecture Overview**](./deep-dive/architecture/overview.md) to get a high-level understanding of the system design and technology stack.

Explore the core concepts and design principles behind EVE-KILL. This section provides a detailed look at the system architecture, data flow, and the technologies that power the application.

### [Development](./deep-dive/development/)
