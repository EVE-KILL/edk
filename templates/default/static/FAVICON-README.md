# Favicon and SEO Images

This directory needs the following image files for proper branding and SEO:

## Required Favicon Files

- `favicon-16x16.png` - 16x16px favicon
- `favicon-32x32.png` - 32x32px favicon (main favicon)
- `apple-touch-icon.png` - 180x180px Apple touch icon
- `android-chrome-192x192.png` - 192x192px Android icon
- `android-chrome-512x512.png` - 512x512px Android icon

## Required SEO Images

- `og-default.png` - 1200x630px Open Graph default image (for social media sharing)

## Design Guidelines

- Use the EVE-KILL logo/branding
- Dark theme (#0a0a0a / #1a1a1a background)
- High contrast for visibility
- Include "EVE-KILL" text or logo

## Tools to Generate

You can use online tools like:

- https://realfavicongenerator.net/
- https://favicon.io/

Or generate manually with ImageMagick/GIMP/Photoshop.
